Eba15 stats rstudio

This repository contains the teaching material for the introductory course NGS Statistics with R, developed for the Aviesan Bioinformatics School (Ecole de Bioinformatique Aviesan) -- Roscoff (France), Sept 27 - Oct 03, 2015

Scope and audience

This course has been conceived to provide life science researchers (biology and biomedical research) with a first contact to statistical analysis of NGS data using the R statistical package.

Practicals

  1. Opening an RStudio session [html] [pdf] [Rmd source]
  2. First steps with R [html] [pdf] [Rmd source]
  3. Basic stats about peak-calling
  4. Exploring RNA-seq counts
  5. Replicates, replicates, replicates, replicates, ...
